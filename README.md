# Socket.IO Real-Time Chat App with Middleware

A simple and easy-to-understand **real-time chat app** built with **Socket.IO**, featuring:

- Private and group chats
- Middleware for authentication and validation
- Clear demonstration of how sockets work
- Perfect for learning real-time communication in Node.js

## Features

- **Real-Time Messaging**: Send and receive messages instantly.
- **Private Chat**: One-to-one messaging between users.
- **Group Chat**: Chat with multiple users in rooms.
- **Middleware Support**: Validate users and events before processing.
- **Learning Focus**: Simplified code to understand Socket.IO fundamentals.

## Tech Stack

- **Node.js** – Backend runtime
- **Express.js** – Web framework
- **Socket.IO** – Real-time communication
- **Middleware** – For socket event validation
